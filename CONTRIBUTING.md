# Contributing Guide

Thank you for your interest in contributing to Nepali Date Picker!

## 🐛 Report a Bug

Found a bug? Please create an issue on GitHub:

1. Go to [Issues](https://github.com/anuzpandey/nepali-date-picker/issues)
2. Click "New Issue"
3. Choose "Bug report" template
4. Describe:
   - What you expected to happen
   - What actually happened
   - Steps to reproduce
   - Your environment (browser, version, etc.)

## ✨ Request a Feature

Have an idea for a new feature?

1. Go to [Issues](https://github.com/anuzpandey/nepali-date-picker/issues)
2. Click "New Issue"
3. Choose "Feature request" template
4. Describe:
   - What the feature should do
   - Why it would be useful
   - Use cases or examples

## 💬 Discussions

- Have questions? Use [GitHub Discussions](https://github.com/anuzpandey/nepali-date-picker/discussions)
- Share ideas and get help from the community
- See how others are using the package

## 📝 Documentation Contributions

Report typos, unclear sections, or suggest documentation improvements:

1. Open an issue describing the problem
2. Or submit a pull request with the fix
3. Help improve examples and guides

## ❓ Questions?

- Check existing [issues](https://github.com/anuzpandey/nepali-date-picker/issues) and [discussions](https://github.com/anuzpandey/nepali-date-picker/discussions)
- Open a new discussion if you have questions
- Be specific and include examples
