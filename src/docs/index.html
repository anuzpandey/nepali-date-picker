Introduction

Nepali Date Picker

Lightweight, Powerful JavaScript Nepali Date Picker with no dependencies.

Meet our user-friendly Nepali Date Picker, tailor-made for Nepal's date format. It's easy to use, doesn't rely on other tools, and makes handling Nepali dates in your web apps a breeze. No more date headaches – just simple and precise Nepali date selection you can count on.

Getting Started

Installation

Download the package

Begin by downloading the Nepali Date Picker package to your local machine. You can find the package on our github repository

Include CSS and JavaScript Files

In your project directory, locate the downloaded package. You will find CSS and JavaScript files.

Copy the CSS file and include it in the <head> section of your HTML file: 

<link rel="stylesheet" href="../css/nepali-date-picker.min.css">

Next, include the JavaScript file at the end of your HTML body:

<link rel="stylesheet" href="../js/nepali-date-picker.min.js">

Initialization

Now, you're ready to initialize the Nepali Date Picker in your JavaScript code.

new NepaliDatePicker({
    format: 'YYYY-MM-DD',
})

CDN Link

To use the Nepali Date Picker via a Content Delivery Network (CDN), you don't need to download any files. Simply include the following CDN links in your HTML file:

<!-- Include the CSS file from the CDN -->
<link rel="stylesheet" href="https://nepali-date-picker.vercel.app/css/nepali-date-picker.min.css">

<!-- Include the JavaScript file from the CDN -->
<script src="https://nepali-date-picker.vercel.app/js/nepali-date-picker.min.js"></script>
